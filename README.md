# Advance-course
This is an Image classification Capstone project.
The dataset is about the Fashinminst:we have to classify the clothes.
About the Dataset:
Fashion-MNIST is a dataset of Zalando's article images, consisting of a training set of 60,000 examples and a test set of 10,000 examples. 
Each example is a 28x28 grayscale image, associated with a label from 10 classes.
Objective: Classify clothes using TensorFlow
We Have used different models in CNN and classified the images accorsingly
The highest accuracy  is 98%
